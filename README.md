## Stack

- Expo ~56 + React Native 0.85 + TypeScript
- expo-router (파일 기반 라우팅)
- GitHub Pages 배포

## Run locally

```bash
git clone https://github.com/Rach0209/miniGames.git
cd miniGames
npm install --legacy-peer-deps
npx expo start --web
```
